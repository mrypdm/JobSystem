## Тема

Разработка системы для защищённого выполнения задач, содержащих произвольный исполняемый код, для ОС Linux

## Структура

### Job.Api

Web API для работы с задачами:
- создание
- запрос результатов

### Job.Worker

Сервис для выполнения задач

### Job.Databse

База данных задач (PostgreSQL).

**Job.Api** добавляет задачи в БД.

**Job.Worker** их оттуда достает для выполнения, а после записывает в БД результаты.

### Job.Broker

Брокер сообщений (Kafka).

**Job.Api** отправляет сообщения о новых задачах через брокер в **Job.Worker**

### User.WebApp

Веб-приложение для конечных пользователей для создания задач и просмотра результатов

### User.Database

База данных пользователей (PostgreSQL).

Содержит авторизационные данные и информацию о созданных задачах
