## Тема

Разработка системы для защищённого выполнения задач, содержащих произвольный исполняемый код, для ОС Linux

## Структура

### Job.Api

Web API для работы с задачами:
- создание
- просмотр результатов

### Job.Worker

Сервис для обработки задач

### Job.Executor

Сервис для выполнения задач

### Job.Watcher

Сервис для очистки старых задач, отмены задач по таймауту и прочих вспомогательных задач
